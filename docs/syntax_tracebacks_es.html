Friendly SyntaxError tracebacks - en español¶

Friendly tiene como objetivo proporcionar comentarios más amigables cuando se produce una excepción se plantea que lo que hace Python (translated by Google Translate).

This file contains only examples of SyntaxError and its sub-classes. Some tests may appear to be repetitive to a human reader but they are may be included to ensure more complete test coverage.

Note

The content of this page is generated by running trb_syntax_spanish.py located in the tests/ directory. This needs to be done explicitly, independently of updating the documentation using Sphinx. On Windows, if Sphinx is installed on your computer, it is suggested instead to run make_trb.bat in the root directory as it will create similar files for all languages and update the documentation.

                                  ^^

renombrado en una línea separada como sigue:

    from math import objeto_1 as nombre_1
    from math import objeto_2 as nombre_2  # si es necesario
